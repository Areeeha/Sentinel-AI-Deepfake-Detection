# Sentinel-AI-Deepfake-Detection
Sentinel is a white-box forensic framework for explainable deepfake detection. It integrates a Neural Engine for texture analysis, a Spectral Pillar to identify FFT-based up-sampling artifacts, and a Geometric Pillar to detect 3D "Perspective Paradoxes." Its novel Veto Step prioritizes physical laws to ensure reliable, auditable verdicts.
